## 分析對象：林口長庚醫院的患者
## 資料集來源：
       林口長庚醫院對於每位患者在1999年-2015年間的健檢資料，但由於在篩選資料上會以較新的資料為主，
    本研究主要將收集2010-2014的資料，研究對象上以IM的病人為主體，IM為腸黏膜化生的縮寫，身為胃黏膜
    病變的一種，常常與腸胃相關疾病有關聯。
## 研究目的：
       希望能透過此研究觀察在對於IM的相關性上哪些變數可能會藉由中介分析從不顯著提升為顯著性，能否與
    參考的文獻有類似的效果或新發現，以及從機器學習的模型檢測哪個組別更顯著。
